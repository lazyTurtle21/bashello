Bashello is a command interpreter.

### Installation :hammer:

1. Clone Project 
2. Compile the Project
```sh
$ mkdir build && cd build
$ cmake -G"Unix Makefiles" ..
$ make
```
3. Run 
```sh
../build$ ./myshell
```
